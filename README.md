# PDF to DOCX GUI

A simple Windows-friendly PDF to DOCX converter with a minimalist black and red cyberpunk UI.

## Features
- Select a PDF file.
- Choose an output folder.
- Convert to DOCX with one click.
- Clean dark UI with red accents.

## Install
```bash
pip install -r requirements.txt
```

## Run
```bash
python -m app.main
```

## Notes
- The app uses PySide6 for native file and folder dialogs.
- Conversion is handled by pdf2docx.
